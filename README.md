# 🛒 Online Marketplace App (Figma Design)

This project is a UI/UX design for an **Online Marketplace Application**, created using **Figma**.  
The app is designed to provide a simple, modern, and user-friendly experience for both buyers and sellers.  

---

## 📌 Project Overview
The Online Marketplace App is a mobile-first design that allows users to:
- Browse products across categories  
- Add items to cart and manage orders  
- Manage your own store (Add, edit or remove products) 
- Provide a secure checkout process  

---

## 🎨 Features in the Design
- **Clean UI:** Minimal and attractive layout for easy navigation  
- **Product Pages:** High-quality product display with detailed descriptions  
- **Shopping Cart:** Smooth cart and checkout flow  
- **User Accounts:** Profile pages for both buyers and sellers  
- **Responsive Layout:** Designed for mobile-first, adaptable for tablets and web
